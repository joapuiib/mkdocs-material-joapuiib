# Magic Links

> Referència: [:octicons-link-external-16: MagicLink &mdash; PyMdown Extensions](https://facelessuser.github.io/pymdown-extensions/extensions/magiclink/)

L'extensió Magic Links està activada per defecte i converteix automàticament les URL en enllaços dins del contingut Markdown.

```yml title="properdocs.yml"
markdown_extensions:
  - pymdownx.magiclink
```

Exemple: https://facelessuser.github.io/
