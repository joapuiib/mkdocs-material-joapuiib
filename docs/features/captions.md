## Captions

Captions can be added to block elements
using the [Caption Block Extension](#).


Generic text
/// caption
Generic Caption
///

![Image](https://via.placeholder.com/150)
/// attribution-caption
Attribution
///
/// figure-caption
Caption for image
///

![Image](https://via.placeholder.com/150)
/// figure-caption | ^1
Virtual nested caption
///

| Header 1 | Header 2 |
|----------|----------|
| Cell 1   | Cell 2   |
/// attribution
Attribution
///
/// table-caption
Caption for table
///
