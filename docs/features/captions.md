## Captions

Captions can be added to block elements
using the [Caption Block Extension](#).


This is the figure.
/// caption
This is the caption.
///

### Image caption

![placeholder](img/placeholder-600x400.png)

///
/// figure-caption
Caption for image
///

![placeholder](img/placeholder-600x400.png)
