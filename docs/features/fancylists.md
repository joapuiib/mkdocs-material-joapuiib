# Fancy Lists

`pymodwn-fancylists` extension is installed by default in this theme. It allows to create
lists with numbers, letters and roman numerals, both uppercase and lowercase.

## List with numbers
2. List item number
3. List item number
4. List item number

```md
2. List item number
3. List item number
4. List item number
```
