# Llistes

> Referència: [:octicons-link-external-16: Lists &mdash; MaterialX](https://jaywhj.github.io/mkdocs-materialx/reference/lists.html)

!!! info
    L'extensió `pymdownx-fancylists` està instal·lada per defecte i permet crear llistes amb números, lletres i numerals romans, tant en majúscules com en minúscules.

## Llistes amb números

2. Element de llista numerat
3. Element de llista numerat
4. Element de llista numerat

```md
2. Element de llista numerat
3. Element de llista numerat
4. Element de llista numerat
```

## Llistes amb lletres

a. Element amb lletra
b. Element amb lletra
c. Element amb lletra


A.  Element amb lletra
B.  Element amb lletra
C.  Element amb lletra

```md
a. Element amb lletra
b. Element amb lletra
c. Element amb lletra

A.  Element amb lletra
B.  Element amb lletra
C.  Element amb lletra
```

També es pot especificar una lletra inicial.

f. Element amb lletra
g. Element amb lletra
h. Element amb lletra

```md
f. Element amb lletra
g. Element amb lletra
h. Element amb lletra
```

## Llistes amb numerals romans

ii. Element romà
iii. Element romà

```md
i. Element romà
ii. Element romà
iii. Element romà
```

## Llistes mixtes

3. Element numèric
4. Element numèric
    a. Element amb lletra
    b. Element amb lletra
        i. Element romà
        ii. Element romà

```md
3. Element numèric
4. Element numèric
    a. Element amb lletra
    b. Element amb lletra
        i. Element romà
        ii. Element romà
```
