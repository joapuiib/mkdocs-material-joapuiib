## Sortable Tables

Use a `div` element with the class `table-sortable` to make a table sortable.

```markdown
/// html | div.table-sortable
| Name | Age | Country |
|------|-----|---------|
| John | 27  | USA     |
| Jane | 23  | Canada  |
| Doe  | 45  | UK      |
///
```
